🗑️ Database Cleanup - Configuration Tables

⚠️ IMPORTANT: Backup apna database pehle! Ye operation irreversible hai.

📋 What will be deleted?

  • tbl_api_configurations - SMTP, Payment, WhatsApp configs
  • tbl_whatsapp_templates - WhatsApp message templates

🔧 Cleanup Methods

Method 1: Using phpMyAdmin (Recommended)

  1. Open phpMyAdmin
  2. Select counselling database
  3. Click on SQL tab
  4. Copy the SQL below and paste it
  5. Click Go
SET FOREIGN_KEY_CHECKS = 0;

DROP TABLE IF EXISTS `tbl_whatsapp_templates`;
DROP TABLE IF EXISTS `tbl_api_configurations`;

UPDATE `tbl_whatsapp_logs` SET `template_id` = NULL, `provider_id` = NULL WHERE 1;
UPDATE `tbl_email_logs` SET `config_id` = NULL WHERE 1;

SET FOREIGN_KEY_CHECKS = 1;

SELECT 'Cleanup completed successfully!' AS Status;

Method 2: Using MySQL Command Line

Windows PowerShell me ye commands run karein:

cd C:\wamp64\bin\mysql\mysql8.0.39\bin
.\mysql.exe -u root -p counselling < "C:\wamp64\www\counselling\counselling-backend\sql\cleanup_quick.sql"

Method 3: Automatic Cleanup (Coming Soon)

✅ Verification

Cleanup ke baad ye queries run karke verify karein:

SHOW TABLES LIKE 'tbl_api_configurations';
SHOW TABLES LIKE 'tbl_whatsapp_templates';

Dono queries empty result return karenge agar successfully delete hui.

🔙 Rollback

Agar problem aaye to backup restore karein:

.\mysql.exe -u root -p counselling < "path\to\backup.sql"
