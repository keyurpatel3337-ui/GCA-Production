# સ્ટાન્ડર્ડ ઓપરેટિંગ પ્રોસિજર (SOP): એકાઉન્ટન્ટ ડેશબોર્ડ કામગીરી

**દસ્તાવેજ ID:** SOP-FIN-002
**આવૃત્તિ:** 1.0
**તારીખ:** 05-Mar-2026
**વિભાગ:** ફાયનાન્સ અને એકાઉન્ટ્સ
**સિસ્ટમ:** જ્ઞાન મંજરી પોર્ટલ

---

## 1. ઝાંખી (Overview)

આ દસ્તાવેજ **એકાઉન્ટન્ટ ડેશબોર્ડ** (`accountant_dashboard.php`) નો ઉપયોગ કરવા માટેની સ્ટાન્ડર્ડ ઓપરેટિંગ પ્રોસિજર (SOP) ની રૂપરેખા આપે છે. ડેશબોર્ડ ફાયનાન્સ સ્ટાફ માટે પ્રાથમિક લેન્ડિંગ પેજ તરીકે કામ કરે છે, જે સંસ્થાકીય નાણાકીય માહિતી, ફી કલેક્શન અને વિદ્યાર્થીઓના પ્રવેશની સ્થિતિનો રિયલ-ટાઇમ સારાંશ પૂરો પાડે છે.

---

## 2. મુખ્ય પર્ફોર્મન્સ ઇન્ડિકેટર્સ (KPIs) અને મેટ્રિક્સ

### 2.1 આંકડાકીય માહિતીની ઝાંખી

ડેશબોર્ડ આપમેળે જટિલ નાણાકીય ડેટા મેળવે છે અને પ્રદર્શિત કરે છે. નાણાકીય લક્ષ્યાંકો પૂરા થાય અને વિસંગતતાઓને દૂર કરવામાં આવે તે સુનિશ્ચિત કરવા માટે કર્મચારીઓએ દરરોજ આ મેટ્રિક્સનું નિરીક્ષણ કરવું જોઈએ.

### 2.2 ઉપલબ્ધ મેટ્રિક્સ અને સંકળાયેલ ક્રિયાઓ

1. **કુલ વિદ્યાર્થીઓ (Total Students)**
   - **વર્ણન:** હાલમાં નોંધાયેલા વિદ્યાર્થીઓની કુલ સંખ્યા દર્શાવે છે.
   - **ક્રિયા:** નોંધાયેલા વિદ્યાર્થીઓની રજિસ્ટ્રી (`students.php?view=enrolled`) પર જવા માટે "View Details" પર ક્લિક કરો.

2. **બાકી ટોકન ફી (Pending Token Fees)**
   - **વર્ણન:** ન ચૂકવેલ રજિસ્ટ્રેશન અથવા ટોકન ફી નું કુલ મૂલ્ય હાઇલાઇટ કરે છે.
   - **ક્રિયા:** નોંધાયેલા વિદ્યાર્થીઓની યાદી જોવા અને ફી કલેક્શન શરૂ કરવા માટે "Collect Fees" પર ક્લિક કરો (`students.php?view=registered`).

3. **બાકી સત્ર ફી (Pending Term Fees)**
   - **વર્ણન:** એકત્ર ન થયેલ શૈક્ષણિક અને હોસ્ટેલ સત્ર ફી કુલ રકમ દર્શાવે છે.
   - **ક્રિયા:** વ્યાપક બાકી પેમેન્ટ રિપોર્ટ (`pending-payments.php`) જોવા માટે "View Details" પર ક્લિક કરો.

4. **સંપૂર્ણ ફી ભરેલ વિદ્યાર્થીઓ (Fully Paid Students)**
   - **વર્ણન:** જે વિદ્યાર્થીઓની કોઈ ફી બાકી નથી તેમની સંખ્યા દર્શાવે છે.
   - **ક્રિયા:** ફાઇલ્ટર કરીને સંપૂર્ણ ફી ભરેલા વિદ્યાર્થીઓની યાદી જોવા માટે "View Details" પર ક્લિક કરો (`students.php?view=enrolled&payment_status=paid`).

5. **માસિક કલેક્શન (Monthly Collection)**
   - **વર્ણન:** વર્તમાન કેલેન્ડર મહિનામાં એકત્રિત થયેલ કુલ આવક દર્શાવે છે.
   - **ક્રિયા:** વિગતવાર નાણાકીય અને કલેક્શન રિપોર્ટ્સ જોવા માટે "View Report" પર ક્લિક કરો (`financial-reports.php`).

6. **કુલ રસીદો (Total Receipts)**
   - **વર્ણન:** સિસ્ટમમાં જનરેટ થયેલી તમામ ફી રસીદોની કુલ સંખ્યા દર્શાવે છે.
   - **ક્રિયા:** નવીનતમ ટ્રાન્ઝેક્શન અને રસીદની હિસ્ટ્રી જોવા માટે કાર્ડ પર ક્લિક કરો (`payments.php`).

## 3. ઝડપી ક્રિયાઓનો વર્કફ્લો (Quick Actions Workflow)

### 3.1 હેતુ

ઝડપી ક્રિયાઓ (Quick Actions) વિભાગ સૌથી વધુ ઉપયોગમાં લેવાતી નાણાકીય કામગીરીઓની વન-ક્લિક એક્સેસ પૂરી પાડે છે, જેથી નેવિગેશનનો સમય બચી શકે.

### 3.2 પ્રક્રિયાઓ

1. **નવી પેમેન્ટ ઉમેરો (Add New Payment)**
   - **ઉપયોગ:** જ્યારે કોઈ વિદ્યાર્થી ફી કાઉન્ટર પર આવે અથવા કોઈ ઓનલાઈન પેમેન્ટ મેન્યુઅલી એન્ટર કરવાનું હોય.
   - **વર્કફ્લો:** **Payments (Add New)** પર ક્લિક કરો -> સિસ્ટમ `add-payment.php` પર લઈ જશે -> સામાન્ય ફી કલેક્શન પ્રક્રિયાને અનુસરો.

2. **નાણાકીય રિપોર્ટ્સ (Financial Reports)**
   - **ઉપયોગ:** દૈનિક, સાપ્તાહિક અથવા મહિનાના અંતે હિસાબના સમાધાન માટે.
   - **વર્કફ્લો:** **Reports (Financial)** પર ક્લિક કરો -> સિસ્ટમ `financial-reports.php` પર લઈ જશે -> જરૂરી નાણાકીય પત્રકો જનરેટ અને એક્સપોર્ટ કરો.

3. **તમામ વિદ્યાર્થીઓ જુઓ (View All Students)**
   - **ઉપયોગ:** વિદ્યાર્થીની પ્રોફાઇલ, નાણાકીય હિસ્ટ્રી અથવા સંપર્ક વિગતો જોવા માટે.
   - **વર્કફ્લો:** **Students (View All)** પર ક્લિક કરો -> સિસ્ટમ વિદ્યાર્થી ડિરેક્ટરી (`students.php?view=enrolled`) પર લઈ જશે.

## 4. મુશ્કેલીનિવારણ અને સિસ્ટમ એલર્ટ્સ (Troubleshooting & System Alerts)

### 4.1 સિસ્ટમ નોટિફિકેશન

- ડેશબોર્ડ વપરાશકર્તાની અગાઉની ક્રિયાના પરિણામના આધારે કલર-કોડેડ એલર્ટ બેનર્સ પ્રદર્શિત કરે છે (સફળતા માટે લીલો, ભૂલો માટે લાલ) (દા.ત., સફળ પેમેન્ટ કલેક્શન અથવા રસીદ જનરેટ કરવામાં નિષ્ફળતા).
- **ક્રિયા:** એલર્ટની નોંધ લો. એલર્ટને ક્લોઝ બટનનો ઉપયોગ કરીને મેન્યુઅલી બંધ કરી શકાય છે.

### 4.2 ડેટા લોડિંગ સમસ્યાઓ (ફોલબેક મોડ)

- **સ્થિતિ:** ડેશબોર્ડના આંકડા અણધારી રીતે તમામ ફીલ્ડ્સ માટે `0` દર્શાવે છે.
- **કારણ:** સિસ્ટમ રિયલ-ટાઇમ ડેટા મેળવવા માટે બેકએન્ડ API (`dashboard/accountant`) પર આધાર રાખે છે. તમામ શૂન્યનું પ્રદર્શન સંભવિત API ટાઇમઆઉટ અથવા કનેક્ટિવિટી સમસ્યા સૂચવે છે.
- **નિરાકરણના પગલાં:**
  1. પેજ રિફ્રેશ કરો (F5 અથવા Ctrl+R દબાવો).
  2. ઇન્ટરનેટ/નેટવર્ક કનેક્ટિવિટી તપાસો.
  3. જો સમસ્યા ચાલુ રહે, તો સિસ્ટમ એડમિનિસ્ટ્રેટર અથવા આઇટી સપોર્ટ ટીમને જાણ કરો, અને જણાવો કે ડેશબોર્ડ API પ્રતિસાદ આપતું નથી.
