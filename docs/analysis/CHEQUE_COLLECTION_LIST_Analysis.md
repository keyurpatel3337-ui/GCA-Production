# PDF Analysis: CHEQUE COLLECTION LIST GM-SGM-MST.pdf

This specific file, **CHEQUE COLLECTION LIST GM-SGM-MST.pdf**, is a specialized view of the transaction history focusing solely on **Cheque payments**. 

Here is a breakdown of its content and how it correlates with your current GCA system:

### 📄 Report Metadata
*   **Period Covered**: 01-01-2026 to 21-02-2026.
*   **Institutions Included**: Gyanmanjari School (GM), Mahatma Seva Trust (MST).
*   **Grouping**: The report is grouped by **Date** and then sub-grouped by the **Educational Unit** (e.g., Gyanmanjari Secondary & Higher Secondary School).

### 📊 Data Columns & Mapping
The PDF contains the following columns, which are fully supported in the GCA **Day Book** and **Receipt Register**:

| PDF Column | GCA System Mapping | Description |
| :--- | :--- | :--- |
| **Rec. No** | `receipt_no` | The unique receipt identifier. |
| **Name** | `student_name` | The full name of the student. |
| **Class** | `current_class` | Student's grade/class (e.g., 12 B GUJ-A). |
| **Bank** | `bank_name` | The bank the cheque was issued from (e.g., UBI, SBI, HDFC). |
| **Cheq. Date** | `cheque_date` | The actual date printed on the physical cheque. |
| **Cheq. No** | `cheque_no` | The 6-digit cheque number. |
| **Amount** | `amount` | The monetary value of the transaction. |

### ✅ Implementation Note
In the GCA system, you do **not** need a separate file for this. This exact report is generated by navigating to the **[Day Book](file:///c:/xampp/htdocs/GCA-Development/portal/modules/reports/financial/day-book.php)** or **Receipt Register** and applying the following filters:
1.  **Payment Mode**: `Cheque`
2.  **Date Range**: Select from/to dates.
3.  **School/Trust**: Select `GM` or `Mahatma Seva Trust`.

The resulting view matches the PDF data structure exactly, including the ability to export to Excel or Print for bank reconciliation.
